What is CSS ?

Cascading style sheets is the real word for CSS. Cascading style sheets are used when a web page is created and CSS is then needed to design the web page to ensure the page looks presentable to the user. The main differences between CSS and html is, CSS is used to style a sheet, but html is used to put specific codes in to ensure the right information is put on to the page. (Html provides structure and CSS provides changes in layout.

What is EcmaScript ?

Scripting – language specification is the real name for ECMAscript. ECMAscript was first published in the year 1997. This was designed by Brendan Eich. This is a scripting language and now is more commonly used for “writing server applications” using Node.js. ECMAscript was made to make Javascript easier to use.
